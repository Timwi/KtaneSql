On the subject of SQL

SQL is a programming language that describes how to manipulate sources of data and generate a difference output source. It is comprised of many different keywords, only a minimal list is used in this module:

SELECT

SELECT X, Y, Z

When you select something, you choose to take all the values that match this X, Y or Z expression and return it. The module offers 3 possible return expressions and they are always simple column references. You must click to select each column to return, you can select between A and G and disable the selection when it shows a "-".

Note: Columns are not always returned in alphabetic order, it could be: A, G then E.

WHERE

WHERE X operation Z

WHERE introduces a filter, all rows will be tested against the conditions you provide the module with and will be included if they match those conditions. The SQL module makes it simple for you, you can only compare columns to fixed values, that means, you can select a column between A and G on the left side and select a value between 0 and 9 on the right side. The possible operators are all simple mathematical operators that you probably already know such as:

  • Equals or "="
  • Not equals or "<>"
  • Less than or "<"
  • Less than or equal or "<="
  • Greather than or ">"
  • Greather than or equal or ">="

If you think you need 2 conditions to achieve the result, click on the button below the WHERE label to introduce a 2nd filter. You can choose between AND and OR to connect the two filters. AND means both conditions must pass to keep the row, OR means at least one condition must pass to keep the row.

Note: You must always specify at least 1 filter.

LIMIT

LIMIT n, o

LIMIT comes with two values. For simplicity, the SQL module splits them for you in two selectors:

  • Limit: Indicates the number of rows to keep from the matching results after apply conditions. If you select "All", no rows are discarded.
  • Skip: Indicates the number of rows to skip after applying conditions. If you select "None", no rows are discarded.
